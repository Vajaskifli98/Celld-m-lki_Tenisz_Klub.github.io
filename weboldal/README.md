## Celldömölki Tenisz Klub Weboldala, adatbázisa és hálozata

A projekt, 3 fős csoportokban készült, 3 fő résszel. 

A weboldal Next.JS-ben készült, Supabase back-endel, a hálozat Cisco packet Tracerben 9.0.0-ás verzióban lett kivitelezve, az adatbázis pedig MySql-ben.

A weboldal kilett deployolva Vercel-re hogy mindenki megtudja nézni (ide kattintva)[https://celldomolki-teniszklub-github-io.vercel.app]
